# ECOMMERSE-SALES-DASHBOARD USING POWER BI

## Project Description
This project focuses on analyzing sales data to understand overall business performance. The main objective is to convert raw sales data into meaningful insights using data analysis and visualization techniques.

---

## Input Data
The project uses two CSV files as input:

### 1. details.csv
Contains sales transaction information:
- Order ID  
- Amount  
- Category  
- Sub-Category  
- Quantity  
- Profit  
- Payment Mode  

### 2. orders.csv
Contains customer and order-related information:
- Order ID  
- Customer Name  
- City  
- State  
- Order Date  

Order ID is used as a common key to connect both datasets.

---

## Process
- Loaded both CSV files into Power BI  
- Established a relationship between the datasets using Order ID  
- Performed basic data cleaning by checking null values and correcting data types  
- Analyzed sales and profit trends  
- Created interactive visualizations for category-wise, sub-category-wise, city-wise, and state-wise performance  
- Added filters and slicers for dynamic data exploration  

---

## Output
The final output is an interactive Power BI dashboard that provides:
- Total sales and profit overview  
- Category and sub-category performance analysis  
- City-wise and state-wise sales distribution  
- Payment mode usage insights  
- Easy data exploration using filters
